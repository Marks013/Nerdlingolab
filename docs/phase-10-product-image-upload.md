# Fase 10: Upload de Imagens de Produto

## Entregue

- Rota autenticada para envio de imagem de produto.
- Validação de tipo e tamanho da imagem.
- Envio para MinIO no bucket configurado.
- Geração de URL pública para salvar no produto.
- Componente de upload no formulário de produto.

## Regras

- Imagens aceitas: JPG, PNG, WebP e GIF.
- Tamanho máximo: 5 MB.
- O campo de URLs permanece como alternativa para importar imagens já hospedadas.
