# Fase 12: Navegação e Layouts

## Entregue

- Cabeçalho público com links para produtos, fidelidade, carrinho e conta.
- Layout administrativo com navegação lateral no desktop.
- Navegação administrativa horizontal no mobile.
- Dashboard simplificado, sem repetir os atalhos já presentes no menu.

## Decisões

- A loja mantém navegação leve e focada em compra.
- O admin prioriza acesso rápido a pedidos, produtos, categorias e cupons.
- A interface segue a política de texto limpo e acentuação correta.
