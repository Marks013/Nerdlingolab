# Fase 8: Admin de Pedidos e Encoding

## Entregue

- Admin de pedidos em `/admin/pedidos`.
- Detalhe do pedido em `/admin/pedidos/[id]`.
- Exibição de status de pedido, pagamento e envio em PT-BR.
- Cards de cliente, endereço, totais, itens e auditoria de estoque/fidelidade.
- Projeto preparado para UTF-8 com `.editorconfig` e `.gitattributes`.
- Documento de regra de acentuação em `docs/encoding.md`.

## Decisões

- Identificadores e código seguem em inglês.
- Texto visível na interface deve usar português brasileiro com acentuação correta.
- A normalização de slug continua removendo acentos apenas para URLs.
