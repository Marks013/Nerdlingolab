# Fase 9: Operação de Pedidos

## Entregue

- Ações administrativas para preparar, enviar, entregar e cancelar pedido não pago.
- Botões operacionais no detalhe do pedido.
- Revalidação das páginas de lista e detalhe após cada ação.
- Textos do detalhe de pedido ajustados para PT-BR com acentuação.

## Regras

- Cancelamento manual só afeta pedidos sem pagamento confirmado.
- Baixa de estoque e fidelidade continuam sendo responsabilidade do pagamento aprovado.
- A interface mostra ações simples para operadores, sem termos técnicos.
